Pooch Memory Match

Visit the Live Site Look at the Code

Pooch Memory Match is a memory match game where a player is shown the backs of 18 cards, 9 of which are matches. The player clicks on one card, is shown the front and then must click on another card in an effort to match it. The player wins when every card is matched.

This is a project on which I worked independently. The primary technology that I focused on was JQuery since this was the quickest way to add user interactivity. The first version was prototyped with Bootstrap for the layout, but I re-did it without Bootstrap and used Flex-box properties instead.

  • JQuery

  • HTML
  • Adobe Illustrator

Student Grade Table

Visit the Live Site Checkout the Code

The student grade table is a table which displays the data of students. A user is able to add students, remove students, update students, and get the latest list of students upon page load or any time a student is added. The formatting and layout was done with Bootstrap and some CSS, the functions were created with JavaScript and JQuery. Originally the Create, Read, Update, Delete funcionality was done with Ajax, but the latest version uses a Firebase Database instead. I had to write and add code in order to perform these operations, some of it is specific to Firebase.

Skills used:

  • Bootstrap
  • Firebase
  • JavaScript
  • JQuery
  • CSS
  • HTML

Meetup Map

Visit the Live Site Look at the Code

This is a web application that I helped to develop with a team of 4 other people in a very short time. A user can enter an interest and a zip code in the form and the most relevant meetups in the nearby vicinity will display on a map. There are also cards containing the information about each meetup displayed on the left side of the screen when on desktop. If mobile, the information can be found in an "info-bubble" on the map. When the user clicks on a card then they are taken to a page with more information about that meetup and videos relevant to their interest.

My part was to make an AJAX call to display videos on the information page. I also made some modifications to the CSS in order to display a little more user friendly on mobile devices. Plus, I used JQuery to modify the application's behavior when on a mobile device.

Skills used:

  • JavaScript
  • JQuery/AJAX
  • CSS
